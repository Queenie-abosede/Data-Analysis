# Los Angeles Crime Analysis
<img width="600" height="350" alt="image" src="https://github.com/user-attachments/assets/8ca16a4e-c5b8-422f-bd79-fd8423ae3625" />


This project analyzes crime trends in Los Angeles from 2020 to the present. Using publicly available LAPD data, it highlights changes in crime pattern. Since 2020, Los Angeles has seen crime trends shift in some surprising ways.This project digs into those changes, looking at everything from neighborhood safety to citywide trends to get a clearer picture of how crime in Los Angeles has evolved over the past few years. The dataset supports exploration, visualization, and modeling to better understand how crime has evolved across the city over time.
